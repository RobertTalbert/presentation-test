How does (discrete) math work?

How many subsets does an $n$-element set have?

$\{0\} \rightarrow \emptyset, \{0\}$; $\{0,1\} \rightarrow \emptyset, \{0\}, \{1\}, \{0,1\}$, etc.

Conjecture: For all $n \in \mathbb{N}$, a set with $n$ elements has $2^n$ subsets.

WHY?

(Induction) The empty set has 1 subset. If a $k$-element set has $2^k$ subsets for some $k$, then a $(k+1)$-element set has $2^k$ subsets with $x_{k+1}$ and $2^k$ subsets without $x_{k+1}$. Hence there are $2^{k+1}$ subsets with $k+1$ elements.
